# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-02-06

### Added
- Initial release of datepainter
- Flexible calendar component supporting multiple integration modes:
  - Astro component integration
  - Vanilla JavaScript integration
  - Template-based rendering for other frameworks
- Built-in state management using nanostores
- Date validation with support for:
  - Day count validation
  - Average window validation
  - Configurable compliance rules
- Theme system with multiple built-in themes:
  - Light theme
  - Dark theme
  - High-contrast theme
- Responsive design with mobile-first approach
- Accessibility features:
  - Keyboard navigation support
  - ARIA labels for screen readers
  - Focus management
- Holiday management system
- TypeScript definitions and full type safety
- Comprehensive styling with CSS variables for customization
- Example implementations for Astro and Vanilla JS
